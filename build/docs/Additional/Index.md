## Installation
PowerAlto is available on the [PowerShell Gallery](https://www.powershellgallery.com/packages/PowerAlto/).

To Inspect:
```powershell
Save-Module -Name PowerAlto -Path <path>
```
To install:
```powershell
Install-Module -Name PowerAlto -Scope CurrentUser
```
