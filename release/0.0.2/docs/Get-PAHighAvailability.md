# Get-PAHighAvailability

## SYNOPSIS
Returns HA information about the desired PA.

## SYNTAX

```
Get-PAHighAvailability [[-PaConnection] <PSObject>]
```

## DESCRIPTION
Returns HA information about the desired PA.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
PAHighAvailability
```

## PARAMETERS

### -PaConnection
